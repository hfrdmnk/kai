What is kai?

kai is a zero-dependency UI annotation tool built as a single Web Component. Developers activate it on any web page to click elements, write feedback, and export structured data for AI coding agents.

The tool runs entirely client-side and outputs JSON or Markdown that includes CSS selectors, computed styles, and human feedback.
